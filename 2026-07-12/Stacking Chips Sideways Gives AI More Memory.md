# Stacking Chips Sideways Gives AI More Memory

- **日期**: 2026-07-08
- **来源**: IEEE Spectrum
- **作者**: Samuel K. Moore
- **链接**: https://spectrum.ieee.org/stacking-chips-sideways
- **分类**: #3D集成 #先进封装 #内存 #AI #HBM

## 摘要

解决一个棘手的3D集成问题可以增加带宽并保持芯片冷却——通过"侧向堆叠"芯片（横向3D集成）为AI提供更多内存。

## 关键点

1. **侧向堆叠方案** — 传统的垂直堆叠面临散热瓶颈，侧向（横向）堆叠芯片可以在不增加热阻的情况下增加内存带宽
2. **带宽提升** — 这种新型3D集成方式显著提高了AI芯片的HBM（高带宽内存）接口带宽
3. **散热优势** — 侧向堆叠避免了垂直堆叠中热量逐层累积的问题，有助于保持芯片工作温度
4. **AI工作负载适配** — AI大模型对内存带宽的需求急剧增长，传统2D布局和垂直堆叠都已达到瓶颈

## 影响分析

"侧向堆叠"代表了3D封装从"叠高楼"到"建社区"的思维转变。散热一直是3D IC最大的痛点，而横向扩展思路可能为高性能AI芯片的热管理提供新的解决路径。这对于HBM和逻辑芯片的集成方案有重大参考价值。
