## 📝 摘要
比利时微电子研究中心（IMEC）在 2025 年 IEEE 国际电子器件会议（IEDM）上研究了将 HBM（高带宽内存）直接堆叠在 GPU 顶部的 3D 封装方案。研究发现，直接堆叠会使 GPU 内部温度翻倍至 140°C，远超 80°C 的工作极限。但通过七项工程优化，团队最终将温差降至接近零，为 3D 芯片集成开辟了可行路径。

## 💡 关键点
- **研究背景**：
  - 当前 2.5D 封装：GPU 和 HBM 并排放置在中介层（interposer）上
  - 2.5D 局限：占用 GPU 两侧空间，限制未来 GPU-GPU 连接
  - 3D 堆叠优势：更高带宽、更低延迟、更小的封装占位面积

- **热挑战**：
  - **2.5D 基准**：GPU 功耗 414W，峰值温度<70°C；HBM 功耗~40W
  - **直接 3D 堆叠**：GPU 温度飙升至 140°C（超出 80°C 限制）
  - 原因：HBM 堆叠阻挡散热路径，热量积聚

- **七项优化方案**：
  1. **移除冗余硅层**：
     - 3D 堆叠后无需 base die 的数据多路复用功能
     - 将内存控制电路移至 GPU 内部
     - 降温效果：~4°C

  2. **降低 GPU 时钟频率**：
     - 3D 堆叠带宽提升 4 倍，LLM 是"内存受限"问题
     - GPU 时钟降速 50% 仍性能领先，降温>20°C
     - 时钟降至 70% 时仅升温 1.7°C

  3. **合并 HBM 堆叠**：
     - 四个堆叠合并为两个更宽的堆叠
     - 消除热量滞留区域

  4. **减薄顶层 die**：
     - HBM 堆叠顶部通常较厚，减薄改善导热

  5. **填充导热硅**：
     - 在 HBM 周围填充更多空白硅块，增强热传导

  6. **优化 HBM 结构**：
     - 提高 HBM 堆叠及周边区域的导热性

  7. **双面冷却**：
     - 传统：95% 热量从顶部通过水冷排出
     - 优化：在底部也添加冷却系统
     - 降温效果：17°C

- **最终结果**：
  - 经过全部优化后，堆叠芯片工作温度降至~70°C
  - 与 2.5D 封装温度相当

- **替代方案**：
  - **GPU-on-HBM**：将 GPU 放在 HBM 上方
  - 优势：GPU 更靠近冷却系统
  - 挑战：GPU 的电源和数据需垂直穿过 HBM，设计更复杂

## 📈 影响分析
**技术意义**：
1. **3D 集成可行性验证**：证明通过系统技术协同优化（STCO），3D 堆叠的热挑战可以克服
2. **带宽突破**：3D 堆叠可提供 4 倍于 2.5D 的带宽，对 AI 计算至关重要
3. **封装小型化**：消除 GPU 两侧占用，为多 GPU 集成腾出空间
