# Obsidian 芯片制造洞察笔记库

## 📖 项目简介

这是一个公开的 Obsidian 笔记库，收录芯片制造产业智能制造相关的权威洞察，包括 Gartner、McKinsey 等咨询机构的报告和分析。

## 🎯 收集范围

### 核心领域
- **智能制造** - 芯片工厂的智能化改造和升级
- **新技术** - AI 质检、数字孪生、预测性维护等
- **新系统** - MES 系统、自动化控制系统等
- **新能力** - 工艺优化、智能调度、质量控制等

### 权威来源
- **咨询机构**: Gartner, McKinsey, BCG, Deloitte, IDC
- **行业组织**: SEMICON, IEEE, SEMI
- **领先企业**: 台积电、中芯国际、华虹、长江存储等

## 🚀 使用方式

### 浏览笔记
1. 按日期浏览对应目录
2. 查看当日的 `📊 芯片制造洞察索引.md` 获取概览
3. 点击具体洞察笔记查看详细内容

### 使用 Obsidian
```bash
1. 安装 Obsidian: https://obsidian.md
2. 打开 Obsidian
3. 选择"打开文件夹"
4. 选择本目录
5. 开始浏览和编辑笔记
```
